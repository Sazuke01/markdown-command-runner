#

## 🚀 Cómo comenzar

1. Instala las dependencias: `pnpm install`.
2. Compila la extensión: ejecuta `pnpm run compile`.
3. Lanza la extensión en VS Code usando la configuración de lanzamiento **Run Extension**.
4. Abre un documento Markdown que contenga un fragmento compatible y haz clic en **Ejecutar** para ejecutarlo.
